1.  INTRODUCTION
-----------------
intranda charts is a small free charting library for bar charts for the 
Java(tm) platform.  It runs on the Java 2 Platform and uses the Java 2D API for 
drawing.

intranda charts is licensed under the terms of the Apache License
Version 2.0. A copy of the licence is included in the

4.  DEPENDENCIES
-----------------
intranda charts has the following dependencies:

(a) JFreeChart - version 1.0.13 or later.  

    http://www.jfree.org/jfreechart/

JFreeChart is licensed under the terms of the GNU Lesser General Public Licence.


(b)  JCommon - version 1.0.0 or later.  

    http://www.jfree.org/jcommon/

JCommon is licensed under the terms of the GNU Lesser General Public Licence.
